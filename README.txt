# nxp-freertos-calibration
Provides Transmit Power Configuration Files for FreeRTOS


This repo, "nxp-freertos-calibration", contains the following Tx Power Limit files for 1ZM, 1XK and 2DS.
   -   wlan_txpwrlimit_cfg_murata_1XK_CA.h    
   -   wlan_txpwrlimit_cfg_murata_1XK_EU.h    
   -   wlan_txpwrlimit_cfg_murata_1XK_JP.h    
   -   wlan_txpwrlimit_cfg_murata_1XK_US.h    
   -   wlan_txpwrlimit_cfg_murata_1XK_WW.h    
   -   wlan_txpwrlimit_cfg_murata_1ZM_CA.h
   -   wlan_txpwrlimit_cfg_murata_1ZM_EU.h
   -   wlan_txpwrlimit_cfg_murata_1ZM_JP.h
   -   wlan_txpwrlimit_cfg_murata_1ZM_US.h
   -   wlan_txpwrlimit_cfg_murata_1ZM_WW.h
   -   wlan_txpwrlimit_cfg_murata_2DS_CA.h
   -   wlan_txpwrlimit_cfg_murata_2DS_EU.h
   -   wlan_txpwrlimit_cfg_murata_2DS_JP.h
   -   wlan_txpwrlimit_cfg_murata_2DS_US.h
   -   wlan_txpwrlimit_cfg_murata_2DS_WW.h


SDK : 2.12.1
============
   These files are already a part of SDK release 2.12.1 and can be found under
Ex: SDK_2_12_1_MIMXRT1170-EVK\components\wifi_bt_module\Murata\tx_pwr_limits
